DABIB

Definitive Artificial Biological Imitation Brain — a biological-inspired memory system for LLMs combining episodic, semantic and procedural memory into a single FastAPI proxy.

What is DABIB?

DABIB (Definitive Artificial Biological Imitation Brain) is an experimental backend that wraps a local LLM server with a brain-inspired memory architecture:

🧠 Episodic memory

Stores raw conversation episodes with timestamps, importance and session metadata. Periodically compressed into concise summaries.

📚 Semantic memory

Long-term facts about the user and the world, distilled during sleep phases and used as context for future answers.

🛠️ Procedural skills

Reusable how-to blocks extracted from interactions and injected when relevant to guide multi-step reasoning and tool use.

How does it work?

The FastAPI app exposes an OpenAI-compatible /v1/chat/completions endpoint. User messages are stored as episodic memories, periodically consolidated into semantic facts and procedural skills. External facts can be lightly verified via a local SearxNG instance. 

# Run the DABIB API
uvicorn app.main:app --host 0.0.0.0 --port 8000

# Configure your OpenAI-compatible client to:
http://localhost:8000/v1/chat/completions
